# Requirements
1. [HuggingFace](https://huggingface.co/) Access Token
2. Python environment with the following packages:
* cmd2
* smolagents==1.9.2
* pandas
* sodapy
* tabulate

# Run
```bash
git clone git@github.com:griffin28/PowerBall.git
export HF_TOKEN=YOUR_HF_TOKEN
cd PowerBall
python app.py
```
