
# PowerBall
AI Agent that generates powerball numbers and displays previous drawings.

# Requirements
1. [HuggingFace](https://huggingface.co/) Access Token
2. Python environment with the following packages:
* cmd2
* smolagents==1.9.2
* pandas
* sodapy
* tabulate

# Run
```bash
export HF_TOKEN=YOUR_HF_TOKEN
python app.py
```
